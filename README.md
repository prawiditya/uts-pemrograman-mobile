## Deskripsi Project
Aplikasi manajemen keuangan berbasis Android Studio menggunakan database SQLite.

## Fitur & UI
1. **Dashboard Utama**: Menampilkan sisa saldo dan daftar transaksi terbaru.
2. **Laporan Keuangan**: Visualisasi data memanjang menggunakan BarChart untuk Pemasukan vs Pengeluaran.
3. **SCRUM Management**: Pengelolaan tugas menggunakan ClickUp.
